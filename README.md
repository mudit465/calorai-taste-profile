# CalorAI Taste Profile

## Overview

A React Native Expo application that allows users to build their taste profile by swiping through food cards. Users can like or dislike foods, track progress, and view a generated taste profile summary.

## Features

- Intro / Onboarding Screen
- Swipeable Food Cards
- Progress Tracking
- Results Screen
- Dark Theme UI
- Food Image Cards
- React Navigation

## Tech Stack

- React Native
- Expo SDK 56
- TypeScript
- React Navigation
- react-native-deck-swiper

## Installation

bash npm install npx expo start 

Press i to launch iOS Simulator.
